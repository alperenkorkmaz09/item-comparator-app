# Item Comparator Application

This is a single-page product comparison application. Users can compare products feature by feature, while the administrator can configure the product database and the maximum number of comparable products.

### Features

- Product catalog with category tabs, search, and dynamic product cards.
- Comparison drawer that shows selected products and enforces the admin-defined limit.
- Side-by-side comparison table for price, brand, category, rating, description, and technical features.
- Difference analysis with highlight differences and show-only-differences options.
- Best value highlight for the lowest price among compared products.
- Admin panel with password login, product create/edit/delete, and comparison limit settings.
- Dynamic feature editor for custom key-value technical specifications.
- JSON tools for database export, import, and reset.
- Light and dark theme support.

### Run

1. Open a terminal in the project folder.
2. Start the server:

```bash
npm start
```

3. Open this address in your browser:

```text
http://localhost:3000
```
